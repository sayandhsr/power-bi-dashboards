# Power BI Dashboards Portfolio

Professional Power BI dashboards showcasing Netflix streaming analytics, Disney+ Hotstar content analysis, and Retail Inventory management with interactive visualizations and business insights.

## Featured Projects

### Netflix Content Analytics Dashboard
Interactive dashboard analyzing 3.3M+ Netflix titles with:
- Content type distribution (Movies vs TV Shows)
- Release year trends (2000s surge)
- Country-wise production analysis
- Rating & genre performance matrix
- Top directors & cast insights

**Source**: netflix_titles.csv (3.3M rows)

### Disney+ Hotstar Content Dashboard
Comprehensive analysis of Disney+ Hotstar catalog featuring:
- Platform vs Release year trends
- Content category breakdown
- Rating distribution heatmap
- Production country insights
- Top performing genres

**Source**: disney_plus_titles.csv (383K rows)

### Retail Inventory Management Dashboard
Real-time inventory tracking with sales analytics, including:
- Top 10 products by quantity sold
- Stock level monitoring (low stock alerts)
- Category-wise performance
- Sales velocity trends
- Supplier performance metrics

**Source**: Inventory-Records-Sample-Data.xlsx
